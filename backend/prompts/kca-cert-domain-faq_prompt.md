**예시:**

## 문서 구조

| 컬럼 | 설명 |
|---|---|
| ID | 고유 행 번호 |

### 계층적 이해
- **업무 대분류** > **업무 중분류** : 문서가 조직별 혹은 기능별로 구분되어 있어 같은 범주 내에서 유사 질문이 군집됩니다.
- 각 Q&A에는 *관련지침·규정* 컬럼으로 제○조, 조항 번호를 직접 명시할 수 있으나 없으면 해당 항목 비고란에 기재됩니다.  
- 별표(금액·비율), 별지(양식) 등이 있을 경우 ‘매뉴얼’ 열에 포함되며, 필요한 경우 답변 시 함께 언급해야 합니다.
