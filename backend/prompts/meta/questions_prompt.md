# 추천 질문 생성

RAG 시스템의 추천 질문을 생성합니다.

## 입력 정보
- 컬렉션: {collection_name}
- 문서 유형: {template_type}
- 문서 샘플:
{document_sample}

## 질문 유형 (4개 생성)
- 정의/개념 1개: "~란?", "~의 정의는?"
- 절차/방법 1개: "~하려면?", "~절차는?"
- 조건/요건 1개: "~조건은?", "~가능한 경우는?"
- 비교/차이 1개: "~와 ~의 차이점은?"

## 규칙
1. 문서에서 답변 가능한 질문만
2. 구체적이고 실용적인 질문
3. 각 질문은 물음표(?)로 끝날 것

## 출력 형식 (필수)
- 반드시 JSON 문자열 배열만 출력
- 코드 블록(```) 없이 바로 출력
- 설명이나 번호 목록 없이 배열만 출력

올바른 출력 예시:
["질문1?", "질문2?", "질문3?", "질문4?"]
