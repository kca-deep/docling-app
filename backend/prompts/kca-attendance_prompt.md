## 문서 구조  
- **장(章)** : 법령 전체를 구분하는 최상위 카테고리.  
- **조(條)** : 각 장 안에서 개별 규정을 정의하는 항목. 예) 제7조, 제12조.  
- **항(①②③)** : 조 내부의 세부 절이나 조건을 나열할 때 사용.  
- **호(1.,2.,3.)** : 항 내에서도 다시 세분화되는 경우 적용.  
- **별표** : 수치·비율·일수 등을 표 형식으로 정리해 두는 부분 (본 샘플에는 없으나 추후 필요 시 참조).  
- **별지** : 서식, 신청서 등 외부 문서를 가리키며 “제○호서식” 형태로 표시됩니다.  
- **단서("다만")** : 특정 상황에서 예외가 되는 부연 설명이 포함된 조항. 반드시 함께 확인해야 합니다.  
- **부칙** : 시행일과 경과처리를 기재하며, 가장 마지막에 위치합니다.
